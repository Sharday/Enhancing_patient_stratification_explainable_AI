Code and data for RISK data normalization and data munging

1. installer.R

Install all R packages

2. COMMANDS_to_generate_ALLRISK.txt

Instructions for data downloaded from GEO and data munging 

3. COMMANDS_to_generate_ALLRISK.sh

Shell script to run all commands, do data download, data munging, etc.

chmod 755 COMMANDS_to_generate_ALLRISK.sh

./COMMANDS_to_generate_ALLRISK.sh

4. risk_data_munging.R

R script for RISK data normalization and data munging
